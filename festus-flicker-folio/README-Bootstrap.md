# Festus Adebayo Portfolio - Bootstrap Version

This is a Bootstrap-based HTML/CSS/JavaScript version of the Festus Adebayo portfolio website, converted from the original React application while maintaining the same look and feel.

## Files Structure

- `index.html` - Main HTML file with Bootstrap structure
- `styles.css` - Custom CSS with design system and animations
- `script.js` - JavaScript for interactivity and functionality
- `placeholder.svg` - Placeholder image (you can replace with actual images)

## Features

### Responsive Design
- Mobile-first approach using Bootstrap 5.3.2
- Responsive navigation with mobile menu
- Adaptive gallery layouts for different screen sizes

### Sections
1. **Navigation** - Fixed navbar with smooth scrolling
2. **Hero Section** - Main title with congratulatory messages
3. **Gallery** - Three sections:
   - Main gallery (45 items in 3-column pattern)
   - UK Government Delegation (20 items)
   - Honorary Fellow Award (19 items)
4. **About** - Bio section with photo
5. **Media Report** - Interactive cards for different topics
6. **Contact** - Contact form with social links
7. **Footer** - Copyright information

### Interactive Features
- Smooth scrolling navigation
- Image modal for gallery items
- Contact form with validation
- Hover effects and animations
- Mobile-responsive navigation
- Notification system for form submissions

### Design System
- Maintains original color scheme and gradients
- CSS custom properties for consistent theming
- Smooth animations and transitions
- Glass-morphism effects with backdrop blur

## Browser Support
- Modern browsers with CSS Grid and Flexbox support
- Bootstrap 5.3.2 compatibility
- ES6+ JavaScript features

## Usage
1. Open `index.html` in a web browser
2. Replace `placeholder.svg` with actual images
3. Update contact form action URL if needed
4. Customize colors and content as required

## Dependencies
- Bootstrap 5.3.2 (CDN)
- Font Awesome 6.4.0 (CDN)
- No build process required - pure HTML/CSS/JS

## Performance
- Optimized for fast loading
- Lazy loading support for images
- Minimal external dependencies
- Efficient CSS and JavaScript

The website maintains the original design aesthetic while being fully functional as a standalone HTML application.
